## Start your dev server

code2vision is **framework agnostic** — it works with React, Vue, Svelte, or any other tool. Just start your dev server as usual:

| Framework | Command |
|-----------|---------|
| Vite (React/Vue/Svelte) | `npm run dev` |
| Next.js | `npm run dev` |
| Create React App | `npm start` |
| Angular | `ng serve` |

Then update the URL in **Settings → code2vision.devServerUrl** if your server runs on a different port.
